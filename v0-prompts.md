## 出店者向けアプリ（store/）

### 1. WelcomeScreen（ログイン/新規登録画面）

```
モバイルファーストのログイン・新規登録画面を作成してください。

要件:
- 393px幅のモバイル画面を基準に設計
- 白背景、アクセントカラーは #5DABA8
- ログイン/新規登録のタブ切り替え
- メールアドレス/パスワード入力フォーム
- LINEログインボタン（緑色 #06C755）
- Googleログインボタン（白背景、グレーのボーダー）
- エラーメッセージ表示エリア
- シンプルで機能重視のデザイン

レイアウト:
- 画面中央に配置されたカード形式
- タブは上部に配置
- フォームは縦に並べる
- ソーシャルログインボタンはフォームの下に配置
- エラーメッセージは赤背景で表示

使用技術: Next.js 14, TypeScript, Tailwind CSS
```

