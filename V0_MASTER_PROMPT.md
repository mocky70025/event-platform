# v0統合プロンプト：Tomorrow プラットフォーム全8画面

このプロンプトを v0 に送信して、Tomorrow プラットフォームの全画面を一括生成してください。

---

## 📋 プロジェクト全体概要

**Tomorrow（トゥモロー）** - イベント出店者、主催者、管理者をつなぐB2Bプラットフォーム

### サービス概要
- **出店者**: イベントを検索し、申し込み、管理
- **主催者**: イベントを作成し、出店者の申し込みを審査
- **管理者**: プラットフォーム全体を統括

## 🏪 出店者アプリ（Store）- 5画面

### 1️⃣ ダッシュボード

**目的**: 全情報を一目で把握

**レイアウト**:
```
┌────────────────────────────────────┐
│ ウェルカムメッセージ + アバター    │
├────────────────────────────────────┤
│ ┌──────┐ ┌──────┐ ┌──────┐      │
│ │応募中│ │承認済│ │出店数│      │
│ └──────┘ └──────┘ └──────┘      │
├────────────────────────────────────┤
│ 🚨 承認完了アラート                │
├────────────────────────────────────┤
│ おすすめイベント（カルーセル）     │
├────────────────────────────────────┤
│ 今後の予定                         │
├────────────────────────────────────┤
│ クイックアクション                 │
└────────────────────────────────────┘
```

**主要コンポーネント**:
- `StatCard`: グラデーションアイコン + 大きな数値 + トレンド
- `AlertBanner`: 4種類（success/warning/info/error）
- `EventCarousel`: 画像 + グラデーション + お気に入りボタン
- `UpcomingEvents`: 日付ブロック + イベント情報

**特徴**:
- Primary color: `#5DABA8`
- hover時: `-translate-y-1 + shadow-lg`
- トランジション: `300ms ease-out`
